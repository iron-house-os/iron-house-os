# Build 95 Pricing Assumptions Policy

## Required assumptions

Every estimate should explicitly state assumptions for:

- Working hours.
- Access constraints.
- Dewatering.
- Shoring.
- Disposal.
- Traffic control.
- Testing.
- Survey/layout.
- Permits.
- Restoration.
- Weather and seasonal constraints.

## Required exclusions

Every bid package should explicitly state exclusions for scopes not priced or not included.

## Review

Assumptions and exclusions must be reviewed before final bid export.
