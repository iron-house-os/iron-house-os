# IHOS MVP Risk Register

| Risk | Impact | Mitigation |
| --- | --- | --- |
| Incorrect takeoff quantity | Bid loss or margin loss | Estimator review required before pricing |
| Default rate not adjusted | Under/over pricing | Review production assumptions per project |
| Missing addendum | Non-compliant bid | Document checklist before final package |
| Supplier quote omitted | Price exposure | RFQ and quote comparison review |
| Unauthorized access | Commercial risk | Keep app internal until auth is hardened |
| File upload error | Missing bid backup | Manual document backup outside IHOS |
| Database loss | Lost estimate/project records | Backups before live use |
