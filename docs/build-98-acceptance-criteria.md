# Build 98 MVP Acceptance Criteria

IHOS reaches first internal MVP acceptance when:

- A project can be created and reopened.
- A takeoff can be summarized and saved.
- Saved takeoffs can be viewed by project.
- Takeoff handoff creates estimate line items.
- Estimate workspace can be saved.
- Saved estimate workspaces can be viewed by project.
- RFQ drafts can be generated.
- Project readiness reflects actual project records.
- Estimate workbook export works.
- Operator can complete the dry run without manual database changes.
