# Known Limitations

## Estimating

- Production rates are defaults and must be adjusted per site.
- Supplier quotes are not automatically imported.
- Estimate workspace reload is stored but not yet fully editable from saved record.

## Takeoff

- PDF/vector geometry parsing is not automated yet.
- Manual quantities and extraction rules require estimator review.
- Saved takeoffs need deeper project tab integration.

## Documents

- Real file upload/storage integration is incomplete.
- Document version control is metadata-based only.

## RFQ

- RFQ drafts are package recommendations, not sent emails.
- Supplier-facing portal is not implemented.

## Security

- Authentication and authorization need to be hardened before external use.
