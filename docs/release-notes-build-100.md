# Release Notes - Build 100 Target

## Major capabilities now in place

- Project workspace foundation.
- Quantity takeoff engine.
- Takeoff persistence.
- Estimate handoff.
- Estimate workspace persistence.
- RFQ linkage drafts.
- Project readiness checks.
- MVP workflow page.
- Project operations page.
- Docker and CI scaffolding.
- Local launch and smoke check documentation.

## Major limitations

- File upload/storage is planned but not fully implemented.
- Authentication/authorization requires hardening before external access.
- Automated drawing geometry takeoff is not implemented.
- Saved estimate workspaces are stored but not fully rehydrated into an editor.

## Intended use

Internal estimator-controlled MVP testing only.
