# Build 226 — Financial control centre

Build 226 establishes the management-only Iron House project financial ledger.

## Delivered

- Convert the latest saved estimate workspace into a project cost budget.
- Track budget, commitments/POs, actual invoices/costs, revenue/progress claims, and forecast adjustments by project and cost code.
- Separate labour, equipment, materials, trucking, subcontract, rental, fuel, overhead, contingency, bonding, insurance, and other costs.
- Calculate forecast cost, cost variance, forecast profit, and forecast margin.
- Capture vendor, PO/invoice/claim reference, date, description, status, and audit author.
- Export non-budget entries as a QuickBooks-friendly CSV for cross-reference.
- Restrict all financial APIs to administrators and operations managers.
- Keep SIN, banking, tax credentials, medical data, and payroll details outside this ledger.
