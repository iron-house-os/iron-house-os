# First Live Bid Checklist

Use this checklist before relying on IHOS for a real estimate.

## Project setup

- Project name, owner, municipality, bid date, and tender number are entered.
- All active drawings/specs/addenda are registered.
- Superseded documents are marked or removed.

## Takeoff

- Drawing scale is confirmed.
- All takeoff quantities have drawing references.
- Low confidence quantities are manually reviewed.
- Saved takeoff appears in project readiness.

## Estimate

- Estimate handoff items are reviewed.
- Self-perform vs subcontract scopes are confirmed.
- Default production rates are adjusted for actual site conditions.
- Supplier pricing is entered or marked as allowance.
- Estimate workspace is saved.

## RFQs

- RFQ drafts match the actual procurement plan.
- Required drawings/specs are attached.
- Wrong-category suppliers are removed.
- Quote due date gives suppliers enough time.

## Final review

- Project readiness has no high-priority blockers.
- Assumptions and exclusions are explicit.
- Bonding, insurance, contingency, overhead, and profit are reviewed.
- Final bid price is independently sanity checked.
