# Build 96 First User Training

## Session 1: Project setup

- Create project.
- Add owner, municipality, bid date.
- Understand project ID.

## Session 2: Takeoff and estimate

- Enter quantities.
- Summarize quantities.
- Save takeoff.
- Send to estimating.

## Session 3: RFQ and readiness

- Generate RFQ drafts.
- Review supplier categories.
- Check project readiness.

## Session 4: Bid review

- Review estimate assumptions.
- Review exclusions.
- Export workbook.
- Compare against manual estimate.
