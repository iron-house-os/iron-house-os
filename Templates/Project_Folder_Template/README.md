# Iron House Project Folder Template

## Standard Folder Structure

```
01_Tender/
02_Drawings/
03_Specifications/
04_Addenda/
05_Municipal_Standards/
06_Site_Visit/
07_Quantity_Takeoff/
08_Estimate/
09_RFQs/
10_Supplier_Quotes/
11_Subcontractors/
12_Schedules/
13_Permits/
14_Construction/
15_Field_Reports/
16_Photos/
17_Quality_Control/
18_Change_Orders/
19_Closeout/
20_Archive/
```

## Folder Purpose
- **01_Tender**: Instructions to bidders, tender forms, bid bond requirements.
- **02_Drawings**: IFC/IFT drawings and revisions.
- **03_Specifications**: Project specifications and referenced standards.
- **04_Addenda**: All issued addenda with tracking log.
- **05_Municipal_Standards**: Applicable MMCD supplements and municipal standards.
- **06_Site_Visit**: Photos, notes, utility locates, and observations.
- **07_Quantity_Takeoff**: Takeoff worksheets and quantity backups.
- **08_Estimate**: Estimate workbook and pricing assumptions.
- **09_RFQs**: Outgoing RFQ packages.
- **10_Supplier_Quotes**: Received quotations and pricing history.
- **11_Subcontractors**: Subcontractor proposals and agreements.
- **12_Schedules**: Bid schedule and construction schedule.
- **13_Permits**: Permit applications and approvals.
- **14_Construction**: Execution documents used after award.
- **15_Field_Reports**: Daily reports and production records.
- **16_Photos**: Progress, QA/QC, and closeout photos.
- **17_Quality_Control**: Testing reports, inspections, deficiencies.
- **18_Change_Orders**: Change requests, pricing, approvals.
- **19_Closeout**: As-builts, warranties, turnover documents.
- **20_Archive**: Final archived bid and project records.

## Naming Convention
`PROJECTCODE_ProjectName_DocumentType_v01.ext`

Example:
`WR26-012_MarineDrive_Estimate_v03.xlsx`

## Operating Rule
Every project begins from this template so that bids, procurement, construction, and closeout all follow the same repeatable structure.
