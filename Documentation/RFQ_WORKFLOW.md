# Iron House RFQ Workflow

## Purpose
Standardize supplier and subcontractor request-for-quotation (RFQ) preparation, distribution, tracking, follow-up, and integration into bid estimates.

## Workflow
1. Identify all supply and subcontract scopes.
2. Select suppliers from the master database.
3. Generate scope-specific RFQ packages.
4. Attach drawings, specifications, and relevant addenda.
5. Send RFQs using standardized branded templates.
6. Record each RFQ in the RFQ log.
7. Follow up before quote due dates.
8. Review received quotations.
9. Enter pricing into the estimate workbook.
10. Archive correspondence and quotes.

## RFQ Package Contents
- Cover email
- Project summary
- Scope description
- Relevant drawings
- Applicable specification sections
- Current addenda
- Requested quote due date
- Delivery location
- Contact information

- Quote Received

- Invalid Contact

## Email Standards
- Professional Iron House branding.
- Include all required attachments.
- Request confirmation if the RFQ reached the wrong recipient.
- Clearly state quote deadline.
- Reference project name and tender number in the subject line.

## Automation Goals
- Generate RFQs directly from estimate scopes.
- Pre-select preferred suppliers.
- Populate attachment lists automatically.
- Track supplier response rates.
- Build pricing history for future estimates.

## Quality Control
Before closing a bid:
- Confirm all critical suppliers were contacted.
- Verify quotes reflect the latest addenda.
- Resolve scope gaps.
- Compare multiple quotes where available.
- Record assumptions for missing pricing.
