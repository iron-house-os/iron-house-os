# Bid Intake and Drawing Review Checklist

Use this checklist when a new civil construction opportunity comes in. It is designed to move a project from raw tender documents into an Iron House estimate, supplier RFQs, assumptions, exclusions, and a go/no-go decision.

## 1. Intake Setup

- [ ] Create project folder.
- [ ] Save tender documents, drawings, addenda, schedules, specifications, and forms.
- [ ] Record project name.
- [ ] Record tender number.
- [ ] Record owner / municipality / authority.
- [ ] Record consultant / engineer.
- [ ] Record closing date and time.
- [ ] Record mandatory meeting date, if applicable.
- [ ] Record site visit requirements.
- [ ] Record bonding, insurance, and WorkSafeBC requirements.
- [ ] Record bid submission method.
- [ ] Record bid validity period.
- [ ] Record required forms and schedules.

## 2. Initial Go / No-Go Screen

- [ ] Location is serviceable.
- [ ] Scope fits Iron House year-one capability.
- [ ] Project size is manageable with available working capital.
- [ ] Schedule is realistic.
- [ ] Bonding or security requirements are manageable.
- [ ] Mandatory experience requirements are manageable.
- [ ] Specialty subcontractor risk is manageable.
- [ ] Traffic-control exposure is manageable.
- [ ] Environmental exposure is manageable.
- [ ] Liquidated damages or delay penalties are acceptable.
- [ ] Payment terms are acceptable.
- [ ] Bid effort is justified.

## 3. Drawing Package Review

- [ ] Confirm latest drawing issue.
- [ ] Confirm drawing index.
- [ ] Confirm civil plan sheets.
- [ ] Confirm profiles and sections.
- [ ] Confirm details and standard drawings.
- [ ] Confirm landscape sheets.
- [ ] Confirm electrical / street-lighting sheets.
- [ ] Confirm traffic-management drawings.
- [ ] Confirm demolition drawings.
- [ ] Confirm erosion and sediment control drawings.
- [ ] Check for missing sheets.
- [ ] Check for unreadable notes or dimensions.
- [ ] Check for conflicting drawing revisions.

## 4. Scope Breakdown

- [ ] Mobilization and demobilization.
- [ ] Layout and survey.
- [ ] Utility locates and daylighting.
- [ ] Traffic control.
- [ ] Environmental controls.
- [ ] Clearing, grubbing, removals, and demolition.
- [ ] Sawcutting and asphalt removal.
- [ ] Concrete removals.
- [ ] Excavation.
- [ ] Trenching.
- [ ] Storm works.
- [ ] Sanitary works.
- [ ] Water works.
- [ ] Manholes and catch basins.
- [ ] Backfill and compaction.
- [ ] Subgrade preparation.
- [ ] Granular sub-base and base.
- [ ] Curb, gutter, sidewalk, letdowns, and concrete flatwork.
- [ ] Asphalt paving.
- [ ] Pavement markings.
- [ ] Signage and traffic furniture.
- [ ] Street lighting and electrical.
- [ ] Landscaping, topsoil, sod, hydroseed, planting, and restoration.
- [ ] Testing and inspection.
- [ ] Closeout and as-builts.

## 5. Quantity Takeoff Checks

- [ ] Compare drawing quantities against measured quantities.
- [ ] Separate self-perform work from subcontracted work.
- [ ] Separate lump sum, unit price, allowance, and optional work.
- [ ] Check excavation depths and trench widths.
- [ ] Check pipe sizes, lengths, classes, and bedding.
- [ ] Check structure counts, sizes, and depths.
- [ ] Check asphalt areas, thicknesses, and tie-in requirements.
- [ ] Check granular areas and depths.
- [ ] Check concrete areas, lengths, thicknesses, and reinforcement.
- [ ] Check topsoil, sod, planting, and restoration quantities.
- [ ] Check disposal quantities and haul distances.
- [ ] Check temporary works and access constraints.
- [ ] Add waste, bulking, shrinkage, and constructability allowances where required.

## 6. Municipal Standards Review

- [ ] Run `MUNICIPAL_STANDARDS_CHECKLIST.md`.
- [ ] Identify applicable municipal supplement.
- [ ] Identify standard details that override generic notes.
- [ ] Confirm approved materials.
- [ ] Confirm testing frequencies.
- [ ] Confirm restoration requirements.
- [ ] Confirm permits and inspections.
- [ ] Confirm traffic-control requirements.
- [ ] Confirm environmental and ESC requirements.
- [ ] Carry cost impacts into estimate.
- [ ] Carry constraints into assumptions and exclusions.

## 7. RFQ Package Requirements

- [ ] Build pipe / utility supplier RFQs.
- [ ] Build structures supplier RFQs.
- [ ] Build aggregate supplier RFQs.
- [ ] Build asphalt subcontractor RFQs.
- [ ] Build concrete subcontractor RFQs.
- [ ] Build traffic-control RFQs.
- [ ] Build testing RFQs.
- [ ] Build coring / sawcutting RFQs.
- [ ] Build trucking / disposal RFQs.
- [ ] Attach relevant drawings, details, and specification pages.
- [ ] Include bid date and quote return deadline.
- [ ] Ask suppliers to redirect if the RFQ reached the wrong inbox.
- [ ] Track sent date, contact, response, quote status, and follow-up date.

## 8. Default Execution Model

Unless the bid requires otherwise, assume Iron House will self-perform:

- Excavation.
- Trenching.
- Pipe installation.
- Manholes and catch basins.
- Backfill and compaction.
- Subgrade preparation.
- Granular sub-base and base.
- Topsoil and cleanup.
- General earthworks using rented equipment with a small in-house crew.

Assume subcontractors for:

- Concrete formwork and placement.
- Fine grading for asphalt.
- Asphalt paving.
- Pavement markings.
- Street lighting and electrical.
- Specialty testing.
- Coring and sawcutting where required.

## 9. Estimate Build Checks

- [ ] Labour crew selected.
- [ ] Equipment selected.
- [ ] Production rates selected.
- [ ] Material costs entered.
- [ ] Supplier quotes entered.
- [ ] Subcontractor quotes entered.
- [ ] Rentals entered.
- [ ] Trucking entered.
- [ ] Disposal entered.
- [ ] Testing entered.
- [ ] Permits entered.
- [ ] Traffic control entered.
- [ ] Environmental controls entered.
- [ ] Overhead entered.
- [ ] Profit entered.
- [ ] Contingency entered.
- [ ] Cash-flow exposure reviewed.

## 10. Constructability and RFI Log

- [ ] Identify missing dimensions.
- [ ] Identify drawing conflicts.
- [ ] Identify utility conflicts.
- [ ] Identify access conflicts.
- [ ] Identify staging constraints.
- [ ] Identify unsuitable material risk.
- [ ] Identify groundwater or dewatering risk.
- [ ] Identify traffic and pedestrian risk.
- [ ] Identify third-party dependency risk.
- [ ] Identify long-lead material risk.
- [ ] Draft RFIs before bid close if required.
- [ ] Convert unresolved items into bid qualifications.

## 11. Bid Submission Review

- [ ] Tender form complete.
- [ ] Schedule of prices complete.
- [ ] Addenda acknowledged.
- [ ] Bonding or bid security included, if required.
- [ ] Insurance documents included, if required.
- [ ] WorkSafeBC clearance included, if required.
- [ ] Subcontractor list complete, if required.
- [ ] Schedule complete, if required.
- [ ] Methodology complete, if required.
- [ ] Assumptions and exclusions complete.
- [ ] Final estimate reviewed.
- [ ] Submission method tested before deadline.
